
# How QR Codes Work — Live Decoder Lab

### 🔗  [see it live](https://elmashaleh.github.io/how-qr-code-work/)

Forget the boring explainer. This one hands you a **real, working QR code** and takes it apart in front of your eyes — scans it, reads its bits live, cracks it open, then rebuilds it back into information, right there on the page.

The code in the hero isn't a stock image or a screenshot. It was built from scratch with a hand-written encoder — Reed-Solomon error correction, masking, the whole spec — and verified to actually scan with a real camera. From there, the lab walks you through *why* QR codes exist, *how* a phone locks onto one in a cluttered photo, *how* black-and-white squares become raw data, *how* the code survives a smudge, a tear, or a logo slapped in the middle, and *where* the decoded payload ends up going.

Every fact, number, and quote is carried over exactly as written in the source paper — this is the science, just made to actually move.

Single HTML file. Zero dependencies beyond Google Fonts. Built to scan.
